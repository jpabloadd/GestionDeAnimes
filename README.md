# Gestor de Animes con API Jikan

Aplicación de **línea de comandos en Python** para gestionar tu catálogo personal de animes.  
Permite **buscar animes automáticamente usando la API de Jikan (MyAnimeList)**, guardarlos localmente, puntuarlos y organizar una lista de espera.

---

# ✨ Características

- 🔎 **Búsqueda automática de animes** usando la API de Jikan
- 💾 **Almacenamiento local en JSON**
- ⭐ **Sistema de puntuación (1-10)**
- 📋 **Catálogo completo en formato tabla**
- ⏳ **Lista de espera para ver más tarde**
- 🖼️ **Visualización de portadas de animes**
- ✏️ **Modo manual si la API no encuentra resultados**

---

# 📦 Tecnologías usadas

- **Python 3**
- **Requests** → consultas HTTP a la API  
- **Pillow (PIL)** → visualización de imágenes  
- **API Jikan** → datos de MyAnimeList

---

# 📁 Estructura del proyecto

- **anime.txt** → se guarda toda la información
- **historial_acciones.txt** → muestra todos los cambios
